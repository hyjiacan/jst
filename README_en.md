# wet

`wet`: `WEb Template`

`wet` is a Server web template engine running on NodeJS, with no 3rd-part dependency.

## Install

Github: https://github.com/hyjiacan/wet

Gitee: https://gitee.com/hyjiacan/wet

npm: https://www.npmjs.com/package/@hyjiacan/wet

```shell
npm i @hyjiacan/wet
```
